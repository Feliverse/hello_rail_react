## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and use as a template, follow these steps.

### Clone <a name="clone"></a>

Clone this repository to your desired folder:

```sh
  https://github.com/Feliverse/hello_rail_react.git
```

To install the necessary dependencies, you should run:

### `gem install bundler`

### `bundler install`

### `yarn install`

## 🤖 Available Scripts <a name="available-scripts"></a>

In the project directory, you can run:

### `rubocop -A`

To fix all Linter error automatically

Star your data base Postgres (Ubuntu):

### `sudo service postgresql start`

To deploy in deployment enviroment:

### `yarn run build`

### `./bin/dev`
